Objects and Classes

In C++ objects are declared as classes, like a blueprint. Here's an example of a base class 



                class Animal {        // The class
                    public:            // Access specifier
                        string kingdom;      // Attribute
                        bool male;          // Attribute
                    public:        // Access specifier
                        void noise();   // Method function declaration
                };


                void Animal::noise() {
                    if(this.male){
                        cout << "rawr |:" << endl;
                    }
                    else{
                        cout << "RAWWWWRRRRR!" << endl;
                    }
                    
                }

Important: Each object's attributes are initialized to a defult value. For bool it's false, and string it's "". If the user doesn't specify a value then the attributes will be set to their respective default value

Constructors!

Think of it like this: When you order a sandwich the constructor is like the worker behind the counter. Instead of having to put the cheese, ham, and lettuce on the sandwich (object) yourself, the constructor does it for you. Without it you'd have have to set each attribute manually. 


                        class Animal {        // The class
                            public:            // Access specifier
                                string kingdom;      // Attribute
                                bool male = false;   // Attribute
                            public:        // Access specifier
                                void noise();
                                Animal(string k, bool m){
                                    kingdom = k
                                    male = m
                                }
                        };

Classes can also have overloaded or multiple constructors as long as they have unique declerations


                        class Animal {        // The class
                            public:            // Access specifier
                                string kingdom;      // Attribute
                                bool male = false;   // Attribute
                            public:        // Access specifier
                                void noise();
                                Animal(string k, bool m){
                                    kingdom = k
                                    male = m
                                }
                                Animal(){
                                }
                        };

Example! 


                    int main(int argc, char *argv[]){
                        Animal animal = Animal();
                        animal.noise()
                        Animal dog = Animal("Animalia", false);
                        dog.noise()
                        
                    }

This example showcases the Animal class' constructor. It set's the male parameter to false which determines the output of the noise function

Inheritance!

Inheritance is where the benefits of object oriented coding shines through, they allow you to define new classes by reusing old classes. This not only keeps you from repeating yourself but elegantly signifies relation between objects. 



                        class Fungus: public Animal{
                            public:
                            Fungus(bool m){
                                kingdom = "fungus"
                                male = m
                            }
                        };

The Fungui class inherits the Animal class but uses a constructor that automatically sets the object's animal attribute to "fungus"
