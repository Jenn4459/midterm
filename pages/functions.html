Functions

Functions are a way to run predefinied code. They increase code reusability by allowing you to rerun large blocks of code with one line, readability by associating a name with a procedure of code, control flow, and iteration through recursion. 



                    // Create a function
                    int double(int a) {
                        return a + a
                    }
                    
                    int main() {
                        // call the function and print it's output
                        output = double(int a)
                        cout <<  output << endl
                    }

Important: Each function has a decleration which you can think of as a contract. The above function is declared as int type, meaning it returns an int. If it had no return type it would ve a void function.

Recursion!

The coolest use of functions is recursion! When a function calls itself within in the function we call that function recursive. const result = fib(4); 


                        int fib(int n) {
                            if (n <= 1) {
                                return n;   // base cases: fib(0)=0, fib(1)=1
                            }

                            return fib(n - 1) + fib(n - 2);
                        }

                        int main(){
                            fib(5)
                        }

When thinking about recursion it's helpful to think from the bottom up. Follow the call stack below to see how the function works up from it's base case (return n)

If a recursive function never reaches its base case i.e return something other than another function call, it will result in an infinite loop
