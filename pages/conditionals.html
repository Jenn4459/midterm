Conditionals

Conditionals allow programs to make decisions. Instead of executing every line of code sequentially, a program can choose different actions depending on whether a condition is true or false.

If Statements

The most common conditional in C++ is the "if" statement. If the condition evaluates to true, the code inside the block executes. 


int age = 18;
if (age >= 18) {
    cout << "You can vote";
}

"You can vote" will be printed by the code above.

If-Else Statements

If the condition is false, the program can execute a different block using the "else" statement. 


int age = 17;
if (age >= 18) {
    cout << "You can vote";
} 
else {
    cout << "You cannot vote";
}

"You cannot vote" will be printed by the code above.

Else If

Multiple conditions can be checked by using multiple "else if" statements together. 


int age = 18;
if (age > 18) {
    cout << "You can vote";
} else if (age < 18){
    cout << "You cannot vote";
} else {
    cout << "Congratulations! You just turned 18! You can now vote!";
}

"Congratulations! You just turned 18! You can now vote!" will be printed by the code above.

Conditional Operators

The operator "==" checks the equality of two variables. "Congratulations! You just turned 18! You can now vote!" will be printed by the code below. 


int age = 18;
if (age == 18) {
    cout << "Congratulations! You just turned 18! You can now vote!";
}

The operator "!=" checks the inequality of two variables. "You are not 18" will be printed by the code below. 


int age = 50;
if (age != 18) {
    cout << "You are not 18";
} else {
    cout << "Congratulations! You just turned 18! You can now vote!";
}
