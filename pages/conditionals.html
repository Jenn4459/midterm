Conditionals

Conditionals allow programs to make decisions. Instead of executing every line of code sequentially, a program can choose different actions depending on whether a condition is true or false.


If Statements

The most common conditional in C++ is the if statement. If the condition evaluates to true, the code inside the block executes. 


                int age = 18;
                if(age >= 18){
                    cout << "You can vote";
                }

If-Else Statements

If the condition is false, the program can execute a different block using the else statement. 


            int number = 10;
            if(number % 2 == 0){
                cout << "Even";
            }
            else{
                cout << "Odd";
            }

Else If

Multiple conditions can be checked by chaining else if statements together. 


                            int score = 85;
                            if(score >= 90){
                                cout << "A";
                            }
                            else if(score >= 80){
                                cout << "B";
                            }
                            else{
                                cout << "C";
                            }
