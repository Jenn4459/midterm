C++ Loops

Often times in programming, we require some type of operation that repeats over and over and over. Computers are well known for doing tasks very quickly, and sometimes, those tasks need to be performed one after another after another.
This is where loops come in. As the name implies, they allow some area of a program to iterate as many times as necessary without needing to code every single line of repetition. It is a powerful tool that greatly increasing the efficiency and readability of code.


Types of Loops

In C++, there are 3 different types of loops: for loops, while loops, and do-while loops


While Loops

While loops have the same structure as conditionals. The only difference is that you use 'while' instead of 'if':

Example 1: 


            int i = 0 ;
            while (i < 5) {
                cout << i << endl;
                i++;
            }
        

            Output:
            0
            1
            2
            3
            4

As you can see from the example, we have initialized a variable i to 0. Then using the while loop, we iterate through the loop until i evaluates to 5.
Note: Once i evaluates to 5, i is checked in the while loop condition and the condition fails, so the loop does not enter. Therefore, the loop ends with the condition check, not at the end of the curly brace.

Important: in this example, the variable used in the loops conditional must increment. Otherwise, you will have an infinite loop!

Example 2:


            int number;
            int guess = 7;

            cout << "Guess the secret number (1-10): ";
            cin >> number;

            while (number != guess) {
                if (number < guess) {
                    cout << "Too low! Try again: ";
                } else {
                    cout << "Too high! Try again: ";
                }
                cin >> number;
            }

            cout << "You guessed the number! Good job!!" << endl;
        

            Sample Output:
            Guess the secret number (1-10): 3
            Too low! Try again: 8
            Too high! Try again: 5
            Too low! Try again: 7
            You guessed the number! Good job!!

This example shows a practical use of while loops: repeating until the user gives the correct input. The loop keeps asking until the condition number != secret becomes false.

Do-While Loops

The key difference from a regular while loop is that a do-while loop always executes its code block at least one time, even if the condition is false.

Example 1:


                int number;

                do {
                    cout << "Enter a positive number: ";
                    cin >> number;
                } while (number <= 0);

                cout << "You entered: " << number << endl;

In this example, Enter a positive number is output before the loop ever runs. This is because of the do{}.

Remember: a while loop checks the condition before entering the loop. A do-while loop will always execute the code block at least once.

For Loops

A for loop is the best choice when you know exactly how many times you want something to repeat.
It 'condition' for the loop has three requirements: initialization, condition, and update. This is best explained with an example

Example 1:


                for (int i = 0; i < 5; i++) {
                    cout << i << endl;
                }
            

                Output:
                0
                1
                2
                3
                4

As you can see, this example produces the exact same output as example 1 of the while loop! It is often a more concise way to write a loop than a while loop, as long as you know exactly how many times the loop will iterate. Now let's go over this example.

Inside the parenthesis of the for loop we have the three conditions:

  • Initialization: This is the int i = 0; It it is possible to omit the int, only if the variable was declared before the loop
  • Condition: This is the terminating condition. It is the exact same as a condition in a while loop.
  • Update: Like in a while loop, you have to update the conditional variable, but with a for loop, you can update it directly in the loops condition!
Note: As you may have noticed, there is a semicolon between each requirement. This is a required element of a for loop. A semicolon must be after the first and second requirement but not the third.

You can also omit requirements when necessary

Example 2:


                for (int i = 0; ; i++) {
                    if (i == 5) {
                        break;
                    }
                }

Or even more extremely!

Example 3:


                for (; ;) {}

It's important to note that this example will produce an infinite loop, as there's nothing to stop it from iterating forever. You'd need a break; statement inside to exit.
