Variables & Data Types

A variable is a named location in memory that holds a value. In C++, you must declare a variable's type before using it.

Common Data Types

  • int – Whole numbers (e.g. 0, 42)
  • double – Decimal numbers (e.g. 3.14)
  • char – A single character (e.g. 'A')
  • bool – True or false
  • string – A sequence of characters (text)

Example

int age = 20;
        double gpa = 3.85;
        string major = "Computer Science";
